# KISSlang

Sample JVM JIT compiler for a sample language created for educational purposes.

# A few examples:
```
# Sample "Hello, world!" program:
PrintLine 'Hello, world!'
```
```
# Some math & variables:

Input line

i=0
If (i==0)
    Print "i is equal to zero"
